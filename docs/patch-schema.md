# Patch schema (V1)

Este documento define o schema de patch da V1 para configuração de módulos DSP.

## Objetivo

- Permitir serialização e restauração de estado.
- Versionar patches para compatibilidade futura.
- Representar parâmetros por `id`, tipo e valor.
