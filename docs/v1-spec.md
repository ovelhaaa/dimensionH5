# V1 modular sandbox spec

Este documento descreve a proposta técnica da V1 para o sandbox modular.

## Escopo

- Arquitetura modular para cadeia de DSP.
- Contrato mínimo de módulos para integração em runtime.
- Fluxo de aplicação de patch por schema versionado.
